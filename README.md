# Portal.Mercado-SemiLivre
Mercado SemiLivre é uma aplicação api inspirada em grandes plataformas de e-commerce como o Mercado Livre, desenvolvida com foco em arquitetura escalável, boas práticas e experiência do usuário.  O projeto simula um marketplace real, permitindo que usuários atuem como compradores e vendedores, com funcionalidades completas de navegação.
